# ИИ-агент на базе фреймворка Agno

## Используемые технологии и инструменты
- **Фреймворк для агентов:** Agno
- **LLM-провайдер:** OpenRouter API 
- **База знаний:** Реализация RAG  с использованием эмбеддингов и векторного индекса для обработки документов
- **Взаимодействие с пользователем:** Telegram Bot API
- **Дополнительно:** Система логирования для отслеживания работы агента и отладки

## Функциональные возможности агента

### 1. Базовая архитектура
Агент функционирует на основе LLM с заданной системной инструкцией, определяющей его поведение и стиль общения. Реализована поддержка диалогового режима.

### 2. Память
Реализован механизм сохранения контекста беседы. Агент способен запоминать информацию, полученную от пользователя в ходе диалога, и использовать её в последующих ответах, что обеспечивает связность взаимодействия.

### 3. Интеграция знаний (RAG)
В проект добавлена функциональность работы с пользовательскими документами:
- Загрузка и обработка текстовых документов
- Генерация эмбеддингов 
- Построение векторного индекса
- Поиск релевантных фрагментов в индексе и формирование ответов на основе найденной информации

### 4. Инструменты
Агент оснащен набором инструментов, которые он может вызывать для выполнения конкретных задач. В рамках курса реализована возможность расширения функциональности агента через добавление пользовательских Python-функций.

### 5. Обработка ошибок и логирование
Внедрена система логирования действий агента, позволяющая отслеживать процесс принятия решений, фиксировать ошибки и анализировать работу системы.

### 6. Интеграция с Telegram
Код агента интегрирован с Telegram Bot API. Реализован запуск бота на хостинге, обеспечивающий возможность круглосуточного взаимодействия пользователей с агентом через интерфейс Telegram.
